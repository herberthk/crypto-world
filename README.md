# CryptoWorld
The home of crypto currencies 
## Getting Started

Clone the project to your local machine

```bash
  git clone https://github.com/herberthk/crypto-world.git
```
Install dependencies by running the following command

```bash
pnpm install
```
Create `.env` file in root directory, add environment variables found in `.env.example` and provide values to them.

Generate environmental variables typings

```bash
pnpm gen-env

# or

npm run gen-env

```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

### Tech-stack
- Nextjs/React
- Typescript
- Ant design UI
- custom css
- Chartjs
- Redux-toolkit/RTK-query
